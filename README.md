# Fireworks

A terminal screensaver that displays animated fireworks with colorful explosions, particle physics, and trail effects.

## Build & Run

```
go build -o fireworks .
./fireworks
```

Press any key to exit.

## Features

- **5 explosion patterns**: Burst, Ring, Willow, Chrysanthemum, Peony
- **7 color palettes**: Classic Red, Gold, Blue Ice, Green, Purple, Rainbow, Silver
- **Particle physics**: Gravity, drag, and quadratic color fading
- **Trail effects**: Particles and rockets leave fading trails
- **Terminal resize**: Adapts to window size changes on the fly

## Requirements

- Go 1.21+
- A terminal with 256-color or truecolor support
